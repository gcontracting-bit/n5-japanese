Grammar Checklist

JLPT N5 Essential Grammar Patterns

▶1. Sentence Structure & Particles

Pattern Meaning Example
は (wa) Topic marker 私は学生です。
が (ga) Subject marker 雨がふっています。
を (wo) Direct object 本を読みます。
に (ni) Location/time/direction 七時に起きます。
で (de) Place of action/means バスで行きます。
へ (e) Direction 学校へ行きます。
と (to) With/and/quotation 友だちと行きます。
も (mo) Also/too 私も行きます。
の (no) Possessive 私の本。
から (kara) From 九時からです。
まで (made) Until 五時までです。
や (ya) And (non-exhaustive) りんごやみかん。

▶2. Copula & Existence

Pattern Meaning Example
です (desu) Is/am/are 学生です。
じゃないです Is not 学生じゃないです。
でした (deshita) Was 学生でした。
あります (arimasu) Exists (things) 本があります。
います (imasu) Exists (people) 猫がいます。

▶3. Verb Forms

Pattern Meaning Example
〜ます Polite present/future 食べます。
〜ません Polite negative 食べません。
〜ました Polite past 食べました。
〜ませんでした Polite past negative 食べませんでした。
〜て (-te form) Connecting 食べて、飲んで。
〜てください Please do 見てください。
〜ないでください Please don't 触らないでください。
〜ています Is doing (progressive) 食べています。
〜ましょう Let's 行きましょう。
〜ませんか Won't you? 行きませんか。
〜たいです Want to 食べたいです。
〜たことがあります Have done 寿司を食べたことがあります。
〜ながら While doing 音楽を聞きながら勉強します。
〜に行きます Go to do 食べに行きます。

▶4. Adjectives

Pattern Meaning Example
い-adj + です Polite present 高いです。
い → くないです Negative 高くないです。
い → かったです Past 高かったです。
い → くなかった Past negative 高くなかったです。
な-adj + です Polite present 静かです。
な-adj + じゃない Negative 静かじゃないです。
な-adj + でした Past 静かでした。
〜のほうが + adj More than これのほうが安いです。
一番 + adj Most 一番大きいです。

▶5. Questions & Requests

Pattern Meaning Example
か (ka) Question marker 学生ですか。
何 (nani/nan) What? 何ですか。
だれ (dare) Who? だれですか。
どこ (doko) Where? どこですか。
いつ (itsu) When? いつですか。
どう (dou) How? どうですか。
どうして Why? どうしてですか。
いくら How much? いくらですか。
いくつ How many? いくつありますか。
どのくらい How long? どのくらいかかりますか。

▶6. Connectors & Other Patterns

Pattern Meaning Example
が (connector) But 日本語は難しいですが、おもしろいです。
から (reason) Because 雨だから、行きません。
ので So/therefore 雨なので、傘を持っていきます。
とき When 子どものとき。
前に Before 寝る前に。
後で After 食べた後で。
と思います I think that おいしいと思います。
でしょう Probably 明日は暑いでしょう。
ね Right? いい天気ですね。
よ Emphasis おいしいですよ。

▶7. Useful Sentence Patterns

Pattern Meaning Example
Nounが好きです Like 音楽が好きです。
Nounが上手です Good at 日本語が上手です。
Nounがあります/います There is 時間があります。
Nounがほしいです Want 水がほしいです。
Nounになります Become 先生になります。
まだ + 〜ていません Haven't yet まだ食べていません。
もう + past Already もう食べました。
